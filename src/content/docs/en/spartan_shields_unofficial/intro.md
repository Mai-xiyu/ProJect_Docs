## ‚ú® Features

- **Diverse Shield Collection**: A wide variety of basic shields and tower shields, from wooden to netherite.
- **Energy Shields**: Shields powered by FE (Forge Energy) and ¬µI (Micro Infinity), no durability repair needed.
- **Block Effects**: Certain shields can apply status effects to attackers upon successful blocking.
- **Tower Shield System**: Large 3D rendered tower shields with greater coverage area and banner pattern support.
- **Rich API**: A complete extension API for developers to easily add custom shields and resource types.
- **Compatibility**: Supports mainstream mods like JEI, EMI, and Curios.

## üõ°Ô∏è Shield Types

### Basic Shield

Standard-sized shields using the durability system, repairable on an anvil. Supports enchantments and banner patterns.

### Tower Shield

Large 3D rendered shields with 1.25x the durability of basic shields. Features larger protection coverage and unique BEWLR rendering effects.

### Energy Shield

Uses FE/RF/¬µI energy instead of durability. Requires charging via chargers (such as Mekanism Energizing Tablet, Thermal Energetic Infuser, etc.). Cannot block when energy is depleted.

## üì• Download & Installation

Supported version: **Minecraft 1.21.1 (NeoForge 21.1.160)**

1. **Download the mod**: Get the latest version from CurseForge or Modrinth.
2. **Place in mods folder**: Put the downloaded `.jar` file into the `.minecraft/mods` folder.
3. **Launch the game**: Ensure the NeoForge loader is installed.

:::note
This is an unofficial port aimed at providing support for newer Minecraft versions.
:::
